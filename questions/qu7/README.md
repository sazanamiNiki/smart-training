## 課題：ピッグ・ラテン (Pig Latin) 翻訳機 🐷

### 概要
英語の文章を受け取り、それを「ピッグ・ラテン」という言葉遊びのルールに従って単語ごとに変換する関数を実装してください。

---
### ピッグ・ラテンのルール

ピッグ・ラテンの変換ルールは、単語が「母音」で始まるか「子音」で始まるかによって決まります。

#### ルールの定義
* **母音**: `a`, `e`, `i`, `o`, `u` (および大文字 `A, E, I, O, U`)
* **子音**: 上記の母音以外のすべてのアルファベット (`b`, `c`, `d`, `f`, ... , `z`, `y`)

---
### 変換ルール

#### 1. 単語が「子音」で始まる場合

1.  単語の先頭から、**最初の「母音（a,e,i,o,u）」または「y」が現れるまで**の連続した子音（これを「子音群」と呼びます）を特定します。
2.  その「子音群」を単語の末尾に移動させます。
3.  その末尾に **"ay"** を追加します。

| 元の単語 | 子音群 | 末尾に移動 | "ay" を追加 | 結果 |
| :--- | :---: | :---: | :---: | :--- |
| **"pig"** | `p` | `igp` | `+ ay` | `igpay` |
| **"smile"** | `sm` | `ilesm` | `+ ay` | `ilesmay` |
| **"string"** | `str` | `ingstr` | `+ ay` | `ingstray` |
| **"rhythm"** | `rh` | `ythmrh` | `+ ay` | `ythmrhay` |
| **"yellow"** | `y` | `ellowy` | `+ ay` | `ellowyay` |


#### 2. 単語が「母音」で始まる場合

1.  単語の末尾に **"way"** を追加します。

| 元の単語 | "way" を追加 | 結果 |
| :--- | :---: | :--- |
| **"apple"** | `+ way` | `appleway` |
| **"eat"** | `+ way` | `eatway` |

---
### あなたのミッション
提供される `main` 関数を完成させ、引数で渡された英語の**文章** (`sentence`) を、単語ごとにピッグ・ラテンに翻訳した新しい文章を返してください。

#### 実装すべきルール
1.  **文章の分割**: まず、文章を**半角スペース** (` `) を区切り文字として、単語に分割する必要があります。
2.  **大文字・小文字の維持**: 元の単語が大文字で始まっていた場合、変換後の単語も大文字で始まる必要があります。
    * 例: `"Pig"` → `"Igpay"`
    * 例: `"Rhythm"` → `"Ythmrhay"`
3.  **句読点の扱い**: 句読点（`,` `.` `!` `?` など）は、翻訳せず、**元の単語の末尾**にそのまま残します。
    * 例: `"Hello,` `world!"` → `"Ellohay,` `orldway!"`

### 入出力の仕様
* **入力**: `sentence: string`
    * 翻訳したい英語の文章。
* **出力**: `string`
    * ピッグ・ラテンに翻訳された文章。

---
### 実装例
```javascript
// 単一の単語（基本）
main("pig")       // "igpay" を返す
main("smile")     // "ilesmay" を返す
main("apple")     // "appleway" を返す

// 句読点と大文字を含む文章（応用）
main("Hello, world!") // "Ellohay, orldway!" を返す
main("What is this?")   // "Atwhay isway isthay?" を返す
main("Rhythm!")       // "Ythmrhay!" を返す
