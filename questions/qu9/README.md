## 課題：テキストの均等割り付け (Text Justification) 📰

### 概要
単語の配列と、1行の最大文字数（`maxWidth`）を受け取り、テキストを「両端揃え（Justify）」で整形して出力する関数を実装してください。

### エラー処理とメッセージ形式 (Error Handling)
* **単語長の超過**:
    * `words` 配列の中に `maxWidth` よりも長い単語が含まれていた場合、即座に例外 (`Error`) をスローして処理を中断してください。
    * **重要**: エラーメッセージには、以下の文字列を含めてください（テストで検証します）。

    `Word "${単語}" (length ${単語の長さ}) exceeds maxWidth (${maxWidth})`

    * 例: 単語 "Hello" (5文字) で maxWidth が 3 の場合
      `Word "Hello" (length 5) exceeds maxWidth (3)`

### ルールと仕様

#### 1. 行への単語の詰め込み（貪欲法）
* 単語は与えられた順序のまま処理します。
* 「単語」と「次の単語」の間には、**最低でも1つの半角スペース**が入ると仮定して計算します。
* その行の文字数が `maxWidth` を超えない範囲で、できるだけ多くの単語を1行に詰め込んでください。

#### 2. スペースの均等割り振り（通常ルール）
行に含まれる単語が決まったら、`maxWidth` に合わせてスペースを配置します。

* **スペースの総数**: `maxWidth` - `(その行の単語の文字数の合計)`
* **配置ルール**:
    * 単語と単語の間の「隙間」に、スペースを可能な限り**均等**に配分します。
    * もしスペースの数が隙間の数で割り切れない場合（余りが出る場合）は、**左側の隙間から順に1つずつ多く**配分します。

#### 3. 特例ルール（左寄せ）
以下の2つのケースでは、上記の「均等割り振り」を行わず、**左寄せ（Left Justified）**で処理します。

* **ケースA: 最後の行 (Last Line)**
    * テキスト全体の最後の行です。
    * 単語の間は**半角スペース1つ**だけで固定します。
    * 行末に残った余白は、すべて半角スペースで埋めます。
* **ケースB: 1行に単語が1つだけの場合**
    * 隙間が存在しないため、均等割り振りができません。
    * 単語の後ろ（行末）に、`maxWidth` になるまでスペースを埋めます。

---
### 入出力の仕様
* **入力**:
    * `words: string[]`
    * `maxWidth: number`
* **出力**:
    * `string[]`

### 実装例

```javascript
// 正常系
main(["This", "is", "an", "example"], 16);
// -> ["This    is    an", "example         "]

// 異常系
main(["Hello", "World"], 3);
// -> Errorがスローされます
// メッセージ内容: 'Word "Hello" (length 5) exceeds maxWidth (3)'
