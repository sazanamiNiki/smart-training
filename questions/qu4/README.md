## 課題：数独チェッカー 🔢

### 概要

9x9の2次元配列で与えられた盤面が、数独のルールとして正しいかどうかを判定する関数を実装してください。

### 数独のルール

数独は、9x9の正方形の盤面を、数字の1から9を使って埋めていくパズルです。正しくないと判定されるのは、以下のいずれかのルールに違反した場合です。

1.  **行 (横)**: 各行（横1列）には、1〜9の数字が重複して出現してはならない。
2.  **列 (縦)**: 各列（縦1列）には、1〜9の数字が重複して出現してはならない。
3.  **ブロック**: 太線で区切られた3x3の各ブロックには、1〜9の数字が重複して出現してはならない。

**この課題では、斜めのルールは考慮しません。**

### あなたのミッション

提供される `main` 関数を完成させ、引数で渡された盤面 (`board`) が、上記の数独のルールをすべて満たしているかどうかを判定し、`true` (正しい) または `false` (間違い) の真偽値を返してください。

#### 判定のポイント
* この関数は、盤面が**ルール違反をしていないか**をチェックするものです。パズルが完全に解けているか（すべてのマスが埋まっているか）や、その盤面が解ける可能性があるかを判定する必要はありません。
* 空のマスは数字の `0` で表現されます。`0` は重複チェックの対象外です。
* 未回答のマス(`0`)と、数字（1〜9）の重複が**両方存在する場合**は、ルール違反とみなし **`false`** を返してください。
    * 例: `[5, 3, 0, 0, 5]` という行は、`0`が含まれていますが、`5`が重複しているためルール違反です。

### 入出力の仕様

* **入力**: `board: number[][]`
    * 9x9の2次元配列。各要素は `0` から `9` までの整数です (`0`は空きマスを表します)。
* **出力**: `boolean`
    * 盤面がルール上正しければ `true`、一つでもルール違反があれば `false`。

### 実装例

```javascript
// 正しい盤面の例 (完成形)
const validSolvedBoard = [
  [5, 3, 4, 6, 7, 8, 9, 1, 2],
  [6, 7, 2, 1, 9, 5, 3, 4, 8],
  [1, 9, 8, 3, 4, 2, 5, 6, 7],
  [8, 5, 9, 7, 6, 1, 4, 2, 3],
  [4, 2, 6, 8, 5, 3, 7, 9, 1],
  [7, 1, 3, 9, 2, 4, 8, 5, 6],
  [9, 6, 1, 5, 3, 7, 2, 8, 4],
  [2, 8, 7, 4, 1, 9, 6, 3, 5],
  [3, 4, 5, 2, 8, 6, 1, 7, 9]
];
main(validSolvedBoard) // true を返す

// 正しい盤面の例 (未完成)
const incompleteBoard = [
  [5, 3, 0, 0, 7, 0, 0, 0, 0],
  [6, 0, 0, 1, 9, 5, 0, 0, 0],
  [0, 9, 8, 0, 0, 0, 0, 6, 0],
  [8, 0, 0, 0, 6, 0, 0, 0, 3],
  [4, 0, 0, 8, 0, 3, 0, 0, 1],
  [7, 0, 0, 0, 2, 0, 0, 0, 6],
  [0, 6, 0, 0, 0, 0, 2, 8, 0],
  [0, 0, 0, 4, 1, 9, 0, 0, 5],
  [0, 0, 0, 0, 8, 0, 0, 7, 9]
];
main(incompleteBoard) // true を返す (重複違反がないため)


// 間違った盤面の例 (未完成:'5' が重複)
const incompleteBoard = [
  [5, 3, 0, 0, 7, 0, 0, 0, 5],
  [6, 0, 0, 1, 9, 5, 0, 0, 0],
  [0, 9, 8, 0, 0, 0, 0, 6, 0],
  [8, 0, 0, 0, 6, 0, 0, 0, 3],
  [4, 0, 0, 8, 0, 3, 0, 0, 1],
  [7, 0, 0, 0, 2, 0, 0, 0, 6],
  [0, 6, 0, 0, 0, 0, 2, 8, 0],
  [0, 0, 0, 4, 1, 9, 0, 0, 5],
  [0, 0, 0, 0, 8, 0, 0, 7, 9]
];
main(incompleteBoard) // false を返す (未完成ではあるが1行目が重複違反なため)

// 間違った盤面の例 (1行目に '5' が重複)
const invalidBoard = [
  [5, 3, 4, 6, 7, 8, 9, 1, 5],
  [6, 7, 2, 1, 9, 5, 3, 4, 8],
  [1, 9, 8, 3, 4, 2, 5, 6, 7],
  [8, 5, 9, 7, 6, 1, 4, 2, 3],
  [4, 2, 6, 8, 5, 3, 7, 9, 1],
  [7, 1, 3, 9, 2, 4, 8, 5, 6],
  [9, 6, 1, 5, 3, 7, 2, 8, 4],
  [2, 8, 7, 4, 1, 9, 6, 3, 5],
  [3, 4, 5, 2, 8, 6, 1, 7, 9]
];
main(invalidBoard) // false を返す
